## Privacy Policy

Last updated: March 4, 2026

### Overview
JurassicMission is a static website. We do not intentionally collect personal
data through forms, accounts, or direct user submissions.

### Data We Process
- Basic web request data may be processed by your hosting provider
  (for example: IP address, user agent, request time, and error logs).
- If analytics tools are added later, this policy should be updated.

### Cookies
This project does not set first-party tracking cookies by default.
Third-party services (such as CDNs) may use their own cookies according to
their policies.

### Audio, Media, and Local Browser Behavior
The site plays media in the browser and may store temporary runtime state in
memory while the page is open. No intentional user profile database is used.

### Third-Party Content
Some content is based on third-party template material under MIT license.
See `THIRD_PARTY_LICENSES.md`.

### Security
Reasonable efforts are made to keep this project safe, but no method of
transmission or storage is 100% secure.

### Contact
For privacy questions, contact the project owner through the repository owner
account on GitHub.
